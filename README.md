# CuStore
Simple e-commerce ASP.NET MVC 5 webite with additional CRM implemented as WCF serivce. 

Used languages, technologies and tools:
- ASP.NET MVC 5 
- WCF
- C#
- EntityFramework 6
- Moq
- Ninject
- JavaScript/JQuery/AJAX
- Bootstrap 3
- log4net
- MSTest

4 projects:
- CuStore.Domain
- CuStore.WebUI
- CuStore.UnitTests
- CuStore.CRMService
